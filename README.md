# C-Project-OpenU
Assembler machine
Semester 2022a - Final Project
### Description

Assembler machine , covert assembly code/file to 16 base code.

### Input

+ assembly code/file

### Output

+ assembly code after deployment marco file
+ 16 base code file
+ external label file
+ data label file

### Methods

+ C
+ Pointers
+ Dynamic Memory Allocation
+ Reading and Creating File
+ MakeFile
